# HMP Asset Manifest

Repository: `geometricmurals/hmp-public-assets`

This repository is the public asset location for approved Hollywood Mural Project support graphics, color values, and usage rules.

## Primary Rules

1. Do not create, redraw, reinterpret, or substitute an AI-generated HMP logo.
2. Use the official Andrew-created HMP logo when available.
3. If the official logo is unavailable, use only the placeholder files in `logo/placeholder/`.
4. Use the icon files in this repository for HMP proposal templates, decks, websites, one-sheets, packets, and AI-generated branded materials.
5. Keep HMP visually distinct from CSS.

## AI Agent Instruction

Before generating final HMP-branded documents, packets, decks, or websites, AI agents should read this manifest and follow the usage rules in `usage_rules/`.

If the final official HMP logo is missing, AI agents must use the placeholder and clearly label it as a placeholder. They must not generate a replacement logo.
