# HMP Public Assets

Public brand asset repository for Hollywood Mural Project (HMP).

This repository contains approved public-facing HMP brand assets and reusable support graphics for AI-generated packets, proposals, decks, websites, and documents.

## Important Logo Rule

This repository may include placeholder logo files for layout testing, but placeholder files are not the official HMP logo.

Use Andrew-created official HMP logo assets only when creating final branded materials.

Do not create, redraw, reinterpret, or substitute an AI-generated HMP logo.

## Asset Categories

- `icons/` — HMP support icons in SVG and PNG sizes
- `logo/placeholder/` — placeholder logo files only
- `colors/` — brand color values
- `usage_rules/` — logo, icon, website, and proposal usage rules

## Rights

These assets are publicly viewable for authorized HMP use. They are not open-source clip art. Do not reuse, modify, sell, or represent these assets without permission from Andrew Severns / Hollywood Mural Project.
