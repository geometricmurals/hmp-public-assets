# HMP Icon Usage Rules

Use the HMP icon set as supporting graphics in proposals, decks, websites, one-sheets, sponsor packets, and AI-generated branded materials.

## Approved Uses

- Benefits sections
- Process diagrams
- Pricing/package cards
- Contact and CTA blocks
- Website feature sections
- Proposal navigation graphics
- Sponsor-facing packets

## Style Rules

- Keep icons small and supportive.
- Do not let icons overpower mural imagery.
- Use icons with generous white space.
- Use the HMP color system consistently.
- Avoid mixing unrelated third-party icon styles in the same document.

## Primary Sprite

Use:

```text
icons/svg/hmp_icon_sprite.svg
```

The sprite contains the corrected proposal-style icon set.
