# HMP Proposal Asset Rules

HMP proposal templates should use this repository for approved public-facing assets.

## Required Asset Checks

Before generating a proposal, deck, one-sheet, or packet, AI agents should check:

1. `ASSET_MANIFEST.md`
2. `colors/hmp_brand_colors.json`
3. `icons/svg/hmp_icon_sprite.svg`
4. `usage_rules/logo_usage_rules.md`
5. `usage_rules/icon_usage_rules.md`

## Proposal Design Rules

- Use clean white backgrounds.
- Use `#111111` for main text.
- Use `#A12767` as the primary accent.
- Use HMP icons sparingly for clarity.
- Use large mural imagery as the main visual driver.
- Use the official HMP logo when available.
- Use the placeholder logo only if the official logo is unavailable.
- Do not generate fake logos.

## Mural Mockup Rule

HMP document branding should follow the HMP brand system, but mural artwork mockups do not need to match the brand colors. Mural mockups should be vibrant, colorful, context-specific, and visually engaging.
