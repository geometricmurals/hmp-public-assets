# HMP Website Asset Rules

Use this repository as the public source for HMP website graphics and brand support assets.

## Recommended Use

- Use SVG icons for scalable web rendering.
- Use the HMP color JSON or CSS variables when building website themes.
- Use the official HMP logo for production websites.
- Use the placeholder logo only during layout drafting when the official logo is unavailable.

## Recommended Icon Sizes

- 24px or 32px for navigation/UI support
- 48px or 64px for feature cards
- 96px or larger for hero/process graphics

## AI Agent Rule

When building or revising an HMP website, first consult:

```text
ASSET_MANIFEST.md
colors/hmp_brand_colors.json
icons/svg/hmp_icon_sprite.svg
usage_rules/logo_usage_rules.md
```

Do not create fake HMP logos.
